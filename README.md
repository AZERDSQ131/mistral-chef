# Mistral Chef 🍳

Transformez vos ingrédients en recettes uniques grâce à l'intelligence artificielle **Mistral AI**.

## Stack

- HTML5 / CSS3 (dark theme, responsive)
- JavaScript vanilla (fetch API Mistral)
- [Mistral AI API](https://docs.mistral.ai/) (mistral-small-latest)

## Fonctionnalités

- Saisie libre d'ingrédients
- 4 ambiances au choix (classique, healthy, exotique, rapide)
- Génération de recette complète : titre, temps, difficulté, ingrédients, étapes, astuce
- Interface dark theme chaude et raffinée
